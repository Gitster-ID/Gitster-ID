Apa itu Git?

Git adalah tools yang berfungsi sebagai Version Control System (VCS) dan kalau diartikan ke bahasa kita artinya sebuah sistem pelacak perubahan pada file. Git dibuat oleh Linus Torvalds. Pada awalnya Git digunakan untuk membantu pengembangan Kernel Linux sebelum akhirnya dipakai lebih luas di berbagai pengembangan software lainnya dan dibuat multi-platform sehingga tersedia di hampir semua OS.

Terlebih dahulu, kamu harus menginstall sebuah Git Client


Linux

apt-get install git
yum install git
zypper in git

Mac

Git untuk Mac OS
brew install git

Windows

Git untuk Windows
